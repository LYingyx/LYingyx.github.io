背景

迭代生成的对抗样本通常过拟合当前网络的结构和权值参数等，因此具有较高的白盒攻击率和较低的黑盒攻击率，而单步生成的对抗样本通常欠拟合当前网络，具有较低的白盒攻击率和较高的黑盒攻击率（相比于用迭代生成的样本来进行黑盒攻击）；如何生成白盒攻击和黑盒攻击成功率都很高的对抗样本？

数据增强可以有效地通过对训练图像应用一组标签保持变换（如调整大小，裁剪和旋转等，即变换完后图像标签不变）来防止网络过拟合，且已有工作证明，图像变换可以防御对抗性示例，即在不同的变换下，对抗样例并不能很好地泛化，这些由对抗性示例经过图像变换产生但并不能成功攻击网络的样本称为硬样本(hard examples)，受到数据增强策略的启发，这些硬样本可以作为新样本来产生转移性更强的对抗性示例（减小了样本对当前网络的过拟合）,这鼓励了我们在迭代生成对抗样本的过程中对样本添加随机且可微的变换（如随机调整大小，随机填充等）（对应于diverse input patterns过程）从而提高样本的可转移性。

I-FGSM在每次迭代时都贪婪地朝着方向添加扰动，这样很容易陷入损失函数局部最小值并且过拟合于特定的网络参数，生成的样本可转移性很差，因此我们需要找到一种更好的优化损失函数的方法来减少这种过拟合现象的发生，如背景中所提到的，我们将在生成对抗样本的过程中随机对图片施加变换以降低过拟合。

(Diverse Inputs Iterative Fast Gradient Sign Method)

指的是在每次迭代时以的概率对样本施加变换来缓解过拟合现象，文章中主要介绍的变换包括随机调整大小(random resizing，将图片的大小调整为随机大小)和随机填充(random padding，在图像周围随机填充零)；此外还有旋转、反转等变换，文中提到经过随机调整大小和随机填充方法生成的对抗样本可转移性越高；

p的大小的确定需要权衡白盒攻击和黑盒攻击的成功率，时退化成，此时白盒攻击成功率高，黑盒攻击成功率低，时，只有经过转换的输入被用于攻击，生成的对抗样本在黑河模型上具有更高的成功率，但在白盒模型上攻击率较低；
